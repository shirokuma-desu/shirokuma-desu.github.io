// Overview

ABOUT THIS PROJECT

Void Protocol started as a solo game jam entry and evolved into a full co-op experience over 6 months. The core loop revolves around entering procedurally generated dungeons, extracting loot, and escaping before the timer runs out — all with up to 3 other players online.

My primary focus was the Gameplay Ability System (GAS) integration, the weapon framework, and the netcode layer ensuring all gameplay actions stayed under 16ms latency on standard connections.

void UWeaponComponent::FireWeapon()
{
    if (!IsValid(WeaponData) || !AbilitySystemComponent) return;

    FGameplayAbilitySpecHandle Handle = AbilitySystemComponent->
        FindAbilitySpecHandleForClass(WeaponData->FireAbilityClass);

    if (Handle.IsValid())
    {
        AbilitySystemComponent->
            TryActivateAbility(Handle, true);
    }
}
